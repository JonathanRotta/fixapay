# 🚀 Fixa Pay - Sistema de Gestão de Pagamentos e Consumo

O **Fixa Pay** é uma solução backend desenvolvida para digitalizar a gestão de pagamentos de pequenos empreendedores. O projeto substitui o controle manual (papel e caneta) por uma estrutura organizada que gerencia diárias, consumações e vales de colaboradores.

---

## 📋 O Problema e a Solução

Atualmente, o fluxo de pagamento é manual, dependendo de assinaturas físicas e anotações que podem se perder ou gerar confusão no fechamento do mês. 

**O Fixá Pay resolve isso através de:**
* **Cadastro Digital de Colaboradores:** Registro de função e valor base da diária.
* **Gestão de Consumo Inteligente:** Registro de itens consumidos no local e vales antecipados.
* **Cálculo Automatizado:** O sistema cruza os dias trabalhados com as consumações pendentes, sugerindo o valor líquido exato para o pagamento.
* **Geração de Comprovantes:** Facilita a prestação de contas entre o empreendedor e o colaborador.
